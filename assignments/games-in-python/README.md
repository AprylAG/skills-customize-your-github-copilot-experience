# 📘 Assignment: Games in Python

## 🎯 Objective

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. Practice string manipulation, loops, conditionals, and random selection.

## 📝 Tasks

### 🛠️ Build the Hangman Game

#### Description
Write a Python program that implements the classic Hangman word-guessing game. The game should manage word selection, track player guesses, and determine win/lose conditions.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses from the player and show current progress in underscore format (_ _ _)
- Track and display the number of incorrect guesses remaining
- End the game when the word is guessed or attempts are exhausted
- Display appropriate win/lose messages
- Allow the player to play multiple rounds if desired
